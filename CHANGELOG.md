### Changed

## [1.0.0] - 2019-06-18

### Added

- Debian base as recommended by DockerHub.
  - "Choose a Secure Base Image" section of [Docker Store Program and Policies](https://success.docker.com/article/store)
- Python 2.7 support
- Database support
  - PostgreSQL
  - MySQL
  - Db2
  - SQLite
